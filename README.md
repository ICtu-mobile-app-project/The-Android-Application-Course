# The-Android-Application-Course
This is a repository which contains the android application project folder, the student grade calculator and a couple of other assignments
1: The ICT Community (Android Development Project)
ICTU Community is a university-ready mobile application built with Kotlin for the frontend, Node.js for the backend/API, and Supabase as the database. The app is designed to serve four types of users: students, teachers, staff, and administrators. It facilitates communication, information sharing, and academic management within the university. Some of the features are: alerts that sends notifications at specific intervals displaying pending or recently created alerts( CA's, exams, assignments etc), the students can also access a database containing a variety of lecture notes, past exam questions etc.
